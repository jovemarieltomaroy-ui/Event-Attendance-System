# Event-Attendance-System
Project Context:
The MSU-IIT DOST Scholars' Association (MIDSA) currently manages large-scale student events using a manual Google Sheets workflow. 
While digital, this current system is highly inefficient as it requires manual data entry that consumes excessive time during high-traffic event check-ins. 
A critical problem is the high risk of untraceable errors; for instance, if an officer accidentally inputs an incorrect student ID number, 
the system provides no immediate way to verify or check who was recently added to the attendance list. 
This lack of real-time validation leads to data integrity issues that are difficult to correct after an event.
To address these challenges, I developed the MIDSA Event Attendance System, a web-based application built with React and MongoDB. 
The proposed solution automates the process through QR code integration, allowing for instant, contactless check-ins that eliminate manual typing errors. 
The system features a centralized Dashboard using NoSQL aggregation to provide live data visualization, such as attendance breakdowns by college and year 
level, giving officers immediate oversight of event participation. Furthermore, the system includes an automated Masterlist for seamless student management 
and a JSON backup feature to ensure data portability and security. The intended users are MIDSA Officers who require a streamlined, accurate, and data-driven 
tool to manage event attendance taking effectively.
